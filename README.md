## Installation

If you want to install this project. You need to use `npm`.

```bash
  npm install

  prisma db push

  prisma generate
```


    
## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`NEXTAUTH_URL`

`DATABASE_URL`

`GITHUB_ID`

`GITHUB_SECRET`
